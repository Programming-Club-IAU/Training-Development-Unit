# Front-end Development Resources

A curated collection of resources to help you learn, practice, and master Front-end Development.

## 1. Learning Roadmaps & Guides
* **[Developer Roadmaps](https://roadmap.sh/frontend)**: The gold standard for visualizing the path from beginner to advanced. It breaks down exactly what skills you need to learn in order.
* **[MDN Web Docs](https://developer.mozilla.org/en-US/)**: The official "handbook" of web development. If you are unsure about a specific HTML tag or CSS property, look here first.
* **[Frontend Masters](https://frontendmasters.com/)**: (Paid) High-quality, deep-dive video courses from industry experts. Excellent for moving from intermediate to advanced.

## 2. HTML & CSS
### Documentation & References
* **[CSS-Tricks](https://css-tricks.com/)**: An incredible archive of articles. Their "A Complete Guide to Flexbox" and "Grid" are legendary.
* **[Can I Use](https://caniuse.com/)**: Essential tool to check if a CSS or JS feature is supported in current browsers.

### Practice & Games
* **[Flexbox Froggy](https://flexboxfroggy.com/)**: A game to learn CSS Flexbox.
* **[Grid Garden](https://cssgridgarden.com/)**: A game to learn CSS Grid.
* **[Frontend Mentor](https://www.frontendmentor.io/)**: They provide professional designs (Figma/Sketch files) and you build them. This is the closest thing to real work experience.

## 3. JavaScript (The Core)
* **[JavaScript.info](https://javascript.info/)**: A very modern, detailed, and easy-to-read textbook on JavaScript.
* **[Eloquest JavaScript](https://eloquentjavascript.net/)**: A classic book that dives deep into the language. Available for free online.
* **[You Don't Know JS (Yet)](https://github.com/getify/You-Dont-Know-JS)**: A book series that explains the "tough parts" of JS (closures, 'this', prototypes) very clearly.

## 4. Frameworks (React, Vue, Svelte)
* **[React Docs (Beta/New)](https://react.dev/)**: The new React documentation is fantastic and teaches "thinking in React" much better than the old docs.
* **[Vue.js Docs](https://vuejs.org/guide/introduction.html)**: Vue is known for having some of the friendliest documentation in the industry.
* **[Svelte Tutorial](https://svelte.dev/tutorial/basics)**: An interactive playground where you learn Svelte by actually writing code in the browser.

## 5. UI & Design Systems
* **[Dribbble](https://dribbble.com/)** & **[Behance](https://www.behance.net/)**: For design inspiration.
* **[Tailwind CSS](https://tailwindcss.com/)**: The most popular utility-first CSS framework.
* **[Google Fonts](https://fonts.google.com/)**: Free typography for your projects.
* **[FontAwesome](https://fontawesome.com/)** or **[Heroicons](https://heroicons.com/)**: Icons for your UI.

## 6. Tools & Environment
* **[VS Code](https://code.visualstudio.com/)**: The industry-standard code editor.
* **[Prettier](https://prettier.io/)**: Code formatter to keep your style consistent.
* **[Git](https://git-scm.com/)**: Version control.
* **[Vite](https://vitejs.dev/)**: A super-fast build tool (a modern replacement for Webpack in many cases).

## 7. YouTube Channels (Free Learning)
* **Traversy Media**: Great crash courses on almost every topic.
* **The Net Ninja**: Excellent playlists on Vue, React, and Flutter.
* **Kevin Powell**: The "King of CSS" on YouTube. If you struggle with CSS layout, watch him.
* **Fireship**: High-intensity, short videos that explain concepts and news very quickly.

## 8. Keeping Up (News & Trends)
* **[Daily.dev](https://daily.dev/)**: A browser extension that gathers dev news.
* **[Hacker News](https://news.ycombinator.com/)**: Tech news (discussion can be advanced, but very insightful).
* **[Smashing Magazine](https://www.smashingmagazine.com/)**: High-quality articles on design and coding best practices.
